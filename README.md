# Folio3D — 3D Portfolio Generator

Upload your LinkedIn or resume PDF and get a beautiful interactive **book-style 3D portfolio** instantly.

## Live Demo
Hosted on GitHub Pages: `https://YOUR-USERNAME.github.io/folio3d`

## Features
- Upload LinkedIn PDF or any resume
- AI parses your profile automatically
- Generates an elegant page-flip 3D book
- Covers: About, Experience, Projects, Education, Skills
- Download 3 files and deploy anywhere free

## How to deploy your generated portfolio
1. Download the ZIP from the app (index.html, style.css, script.js)
2. Drag the folder onto [Netlify Drop](https://app.netlify.com/drop)
3. Get a permanent free URL instantly

## Tech
Vanilla JS · CSS 3D transforms · PDF.js · Claude AI · No frameworks
